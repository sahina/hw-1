# AMEX Assignment

## Requirements

* `node.js` (at least version 6.x)
* `npm` package manager
* postgres database
* `yarn` package manager

## Steps to run locally

Clone repository

    git clone git@github.com:sahina/hw-1.git

Install dependencies

    yarn install
